HTML elementi

Šodien mēs jau iemācījāmies kas ir HTML elementi un kā ar tiem darboties.
Mēs jau esam paspējuši apskatīt tādus HTML elementus, kā:


  • Konteiners "< div >"
  • Virsraksts "< h1 >, < h2 >, ..., < h6 >"
  • Rindkopa "< p >"
  • Atribūts "< a >"
  • Rindas atdalītājs "< br >"

Tagad apskatīsim vēl šos elementus


  1. Nenumurēts saraksts "< ul >" aļa unordered list
  2. Saraksta objekts " < li > "
  3. Numurēts sakārtots saraksts " < ol >" aļa ordered list
  4. Attēlu tags "< img >"
  5. Poga "< button >"
  6. Treknraksts "< b >"
Vairāk informācijas par HTML elementiem mēs varam atrast w3schools mājaslapā.
